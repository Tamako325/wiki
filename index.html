WaterHill + CC Switch + Codex

Codex 接入中转站使用 GPT 教程

按照三步完成配置：先在 WaterHill 中转站创建 GPT 分组密钥，再用 CC Switch 添加 OpenAI 兼容供应商，最后安装 Codex 并切换到刚配置好的供应商。

配置路径

  1. WaterHill创建 GPT 分组 API 密钥
  2. CC Switch添加 OpenAI 兼容供应商
  3. Codex安装后使用该供应商

开始前

你需要准备什么

一个可访问的浏览器

用于注册中转站、下载 CC Switch，以及打开 Microsoft Store 页面。

一个安全保存密钥的位置

API 密钥创建后建议立即保存，不要公开发送给他人。

Windows 设备

本教程的 Codex 下载入口使用 Microsoft Store 链接，适合 Windows 用户。

第一部分

注册 WaterHill 中转站

打开 https://www.waterhill.cyou/， 点击“立即开始”或右上角“登录”都可以进入登录页面。没有账号时点击“注册”，优惠码可以不填。

第一部分

创建 API 密钥、选择分组并复制密钥

1

进入 API 密钥页面

登录后，在左侧菜单中点击 API 密钥。

2

点击创建密钥

在 API 密钥页面点击右上角“创建密钥”，准备进入分组选择流程。

3

选择 GPTfree 和混合分组

进入选择分组页面后，主分组和副分组分别选择 GPTfree 和 混合 即可，一个选 GPTfree，一个选混合。

4

复制生成的密钥

密钥创建成功后，回到 API 密钥列表，点击复制按钮保存密钥。后面配置 CC Switch 时会用到它。

第二部分

下载 CC Switch

优先打开 CC Switch 官网下载。如果官网无法下载，打开 GitHub Releases 页面，选择适合 Windows 的安装包或压缩包。

官网下载 https://www.ccswitch.io/zh/ GitHub Releases 备用下载 无法从官网下载时使用

第二部分

在 CC Switch 添加供应商

打开 CC Switch，在 OpenAI 或 Codex 相关配置界面中点击右上角“+”添加供应商。

第二部分

填写供应商参数

名称可以自己填写，API 请求地址填写 https://www.waterhill.cyou/v1， 密钥填写第一部分创建的 WaterHill API 密钥，模型名称填写 gpt-5.5。

第三部分

下载 Codex

可以直接在 Microsoft Store 搜索下载 Codex，也可以打开下面的 Microsoft Store 链接安装。

Microsoft Store 下载页 安装后回到 CC Switch 选择刚添加的 WaterHill 供应商

参数速查

CC Switch 填写内容

字段 填写内容
名称 自定义，例如 WaterHill GPT
API 请求地址 https://www.waterhill.cyou/v1
API 密钥 WaterHill 网站 API 密钥页面创建的密钥
分组 GPT
供应商或模型 gpt-5.5
